
Die Modellierungen im rdf/xml Format können in einem normalen Editor (zb. http://notepad-plus-plus.org/) geöffnet werden.

Um sie in einer übersichtlichen Baumstuktur zu betrachten kann Protégé (http://protege.stanford.edu/) runtergeladen werden 
und die Datei darin geöffnet werden.

In den pdf Dateien finden sich die rdf/xml Dateien als pdf. Dabei besteht aber keine Garantie für die optische Darstellung.

Graphml Dateien müssen im yEd Graph Editor (http://www.yworks.com/en/products/yfiles/yed/) geöffnet werden. 
Die grafische Abbildung kann aber auch in der png Datei angeschaut werden.

Sämtliche Dateien welche sich in Archivordnern befinden wurden nur vollständigkeithalber hinzugefügt. 
Es besteht kein Anspruch auf Korrektheit dieser Dokumente.
